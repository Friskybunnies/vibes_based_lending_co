# Vibes-Based Lending Co.

Simple full-stack app using React + Express.

## What it does
- Form in React → Express → Alloy sandbox.
- Result screen depends on the evaluation outcome (Approved / Manual Review / Denied).

## How to run

### Backend
```bash
cd backend
npm install
node index.js
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```
